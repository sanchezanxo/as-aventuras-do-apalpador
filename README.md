# As Aventuras do Apalpador

Xogo de plataformas 2D en galego protagonizado polo Apalpador, a mítica personaxe do Nadal galego.

## Xogar

Visita [apalpador.io](https://apalpador.io) para xogar directamente no navegador.

## Controis

- **Frechas esquerda/dereita**: Mover
- **Frecha arriba / Espazo**: Saltar (preme dúas veces para dobre salto)
- **Paredes**: Pégache ás paredes e salta delas

## Características

- Plataformas 2D estilo pixel art
- Dobre salto e wall-jump
- Colecciona castañas
- Sistema de vidas e tempo
- Trampas e obstáculos
- Interface en galego

## Tecnoloxía

- [Phaser 3](https://phaser.io/) - Framework de xogos HTML5
- JavaScript ES6+
- HTML5 / CSS3
